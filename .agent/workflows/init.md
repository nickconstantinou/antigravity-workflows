# Workflow: Project Initialization
1. **PRD Generation:** Analyze my prompt and create a `PRD.md` in the root.
2. **Environment Setup:**
   - Install MCP servers for Supabase and Railway if missing.
   - Run `supabase init` and `railway login`.
   - Install Supabase & Github CLI
3. **Repository:** Initialize `git`, create a GitHub repo using `gh repo create`, and push the initial structure.
4. **Plan:** Present a walkthrough of the proposed folder structure for approval.
